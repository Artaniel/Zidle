# 1. Вирусная эволюция (Система перков)

## 1.1. Магазин перков
- [ ] Панель с кнопками перков (аналогично магазину)
- [ ] На каждой кнопке отображается:
  - Иконка перка
  - Название
  - Текущий уровень (1-3)
  - Стоимость следующего уровня в очках мутаций
  - Краткое описание эффекта
- [ ] При клике на кнопку:
  - Если хватает очков мутаций - перк улучшается
  - Списываются очки мутаций
  - Обновляется описание (если эффект зависит от уровня)
  - Обновляется стоимость следующего уровня

## 1.2. Перки заражения
- [ ] Ускоренное заражение: +20% к скорости инкубации вируса
- [ ] Воздушно-капельный путь: вирус может передаваться по воздуху
- [ ] Кровяной контакт: +50% к шансу заражения при атаке
- [ ] Устойчивость: снижает эффективность лечения на 30%
- [ ] Скрытое течение: увеличивает время до появления симптомов

## 1.3. Перки зомби
- [ ] Усиленные мускулы: +30% к урону
- [ ] Крепкая шкура: +30% к здоровью
- [ ] Быстрые ноги: +20% к скорости передвижения
- [ ] Взрывная смерть: при смерти зомби взрывается, нанося урон
- [ ] Ядовитое облако: зомби оставляет за собой ядовитый след

## 1.4. Интерфейс магазина перков
- [ ] Панель с кнопками перков (3-4 колонки, скролл при необходимости)
- [ ] Фильтры по категориям:
  - Все перки
  - Заражение
  - Характеристики зомби
  - Особые способности
- [ ] Сортировка:
  - По стоимости
  - По названию
  - По силе эффекта
- [ ] Индикатор доступных очков мутаций в верхней части экрана
- [ ] Всплывающая подсказка при наведении на перк:
  - Подробное описание
  - Текущий уровень
  - Эффект следующего уровня
  - Стоимость улучшения

# 2. Механика заражения

## 2.1. Основные параметры
- [ ] Скорость заражения (от 0.1 до 1.0)
- [ ] Радиус заражения (1-5 метров)
- [ ] Шанс заражения при контакте (10-100%)
- [ ] Время инкубации (10-60 секунд)

## 2.2. Способы передачи
- [ ] Прямой контакт (атака зомби)
- [ ] Воздушно-капельный путь (в радиусе)
- [ ] Через кровь (после смерти зараженного)
- [ ] Через воду/еду (специальные здания)

## 2.3. Визуальные эффекты
- [ ] Анимация заражения человека
- [ ] Визуальные индикаторы зараженных
- [ ] Эффекты вокруг зомби (ядовитый туман и т.д.)
- [ ] Изменение внешности при заражении

# 3. Система ресурсов

## 3.1. Основные ресурсы
- [ ] Очки мутаций (за заражение людей)
- [ ] Биомасса (за убитых зомби)
- [ ] ДНК-образцы (редкий дроп с особых целей)

## 3.2. Получение ресурсов
- [ ] Выпадение с зараженных людей
- [ ] Уничтожение зданий
- [ ] Ежедневные награды
- [ ] Достижения

## 3.3. Интерфейс ресурсов
- [ ] Отображение текущего количества
- [ ] Анимация сбора
- [ ] Всплывающие сообщения о получении
- [ ] Звуковые эффекты сбора
